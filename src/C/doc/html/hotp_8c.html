Génère un mot de passe à partir du secret en utilisant un compteur. D'après l'algorithme donné dans la rfc 4226.

Paramètres
[in]keyle secret utilisé pour générer un OTP.
[in]counterle compteur permettant de générer un OTP.
[in]lenla longueure de l'OTP a générer.
Précondition
secret != NULL.
counter > 0.
len >= 6.
len <= 8.
Renvoie
l'entier correspondant à l'OTP; -1 en cas d'échec
