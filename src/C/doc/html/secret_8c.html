secret createRandomSecret ( int  length)

Crée un "objet" secret rempli avec des length octets aléatoire. Cette fonction utilise le fichier /dev/random pour générer de l'aléatoire. Elle est donc plus fiable mais beaucoup plus lente que createSecret.

Paramètres
[in]lengthla taille du secret en octets.
Précondition
la taille ne peut pas être négative.
Renvoie
L'adresse d'un nouveau secret.

secret hexToSecret ( char *  buffer)

Créée un secret à partir d'une chaîne de caractères héxadécimaux.

Paramètres
[in]bufferle pointeur sur le buffer contenant la chaîne à transformer
Précondition
buffer != NULL
buffer[strlen(buffer) - 1] == 0.
strlen(buffer) > 0
pour 0 <= i < strlen(buffer):
'0' <= buffer[i] <= '9'
|| 'a' <= buffer[i] <= 'f'
|| 'A' <= buffer[i] <= 'F'
Renvoie
un secret initialisé avec le buffer; NULL en cas d'échec.

Un pointeur temporaire sur un buffer contenant une chaîne héxadécimale de longueur paire équivalent a celle contenue dans buffer
