secret createRandomSecret ( int  length)

Crée un "objet" secret rempli avec "length" octets aléatoires.

Cette fonction utilise le fichier /dev/random pour générer de l'aléatoire.
Elle est donc plus fiable mais beaucoup plus lente que createSecret.

Paramètres
[in]lengthla taille du secret en octets.
Précondition
la taille ne peut pas être négative.
Renvoie
L'adresse d'un nouveau secret. NULL en cas d'erreur.

int destroySecret ( secret  key)

Libère les ressources associés à un secret.

Cette fonction désallouera les ressources mémoire correspondant aux octets du secret ainsi qu'à la structure secret.
Par mesure de sécurité ces octets auront été mis à zéro avant libération.

Paramètres
[in]keyle secret dont on veut libérer les ressources.
Précondition
key != NULL
Renvoie
0 si les ressources ont été libérées; -1 sinon.
char* getHexRepresentation ( secret  key,
char *  buffer,
int  length 
)

Donne une représentation hexadécimale du secret.

Cette fonction va remplir le buffer passé en paramètre avec length octets afin de donner une représentation hexadécimale du secret.
Si length octet ne sont pas suffisant pour représenter le secret alors la fonction renverra une erreur.

Paramètres
[in]keyle secret que l'on veut représenter.
[out]bufferle buffer destiné à contenir la représentation du secret.
[in]lengthle nombre d'octet modifiables dans le buffer.
Précondition
key != NULL
buffer != NULL.
taille de buffer >= length.
length > 0
length > (key->length * 2)
Postcondition
buffer contient une représentation du secret.
strlen(buffer) <= length.
Renvoie
l'adresse de buffer; NULL en cas d'échec.

secret hexToSecret ( char *  buffer)

Créée un secret à partir d'une chaîne de caractères héxadécimaux.

Cette fonctions va allouer un espace mémoire et convertir chaque paire de caractères héxadécimaux dans le buffer passé en paramètre en un octet qu'elle gardera dans l'espace alloué.
Si il y a un caractère non héxadécimal dans la chaine buffer alors la fonction retournera une erreur.

Paramètres
[in]bufferle pointeur sur le buffer contenant la chaîne à transformer
Précondition
buffer != NULL
buffer[strlen(buffer) - 1] == 0.
strlen(buffer) > 0
pour 0 <= i < strlen(buffer):
'0' <= buffer[i] <= '9'
|| 'a' <= buffer[i] <= 'f'
|| 'A' <= buffer[i] <= 'F'
Renvoie
un secret initialisé avec le buffer; NULL en cas d'échec.
secret textToSecret ( char *  buffer)

Crée un secret à partir d'une chaîne de caractères.

Cette fonction va utiliser les octets pointés par buffer comme nouveau secret.
Pour cela elle va allouer un nouvel espace en mémoire et recopier les octets pointés par buffer.

Paramètres
[in]bufferle pointeur sur la chaîne de caractères.
Précondition
buffer != NULL
strlen(buffer) > 0
buffer[strlen(buffer) - 1] == 0
Renvoie
un secret initialisé avec la chaîne de caractères pointée par buffer.
