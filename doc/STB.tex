\section*{Objet}
Etude et implémentations des systèmes d'auth via mdp jetable :
\begin{itemize}
Besoin op.
\begin{itemize}
\item etat de l'art sur le syst existant
\item fournir un logiciel exploitable implémentant ce syst
\item objectif : mise en prod
\end{itemize}
Obj tech
\begin{itemize}	
\item mise en place état de l'art sur certain (voir tous les) syst
\item Implémentation
\end{itemize}
Contraintes et recom :
\begin{itemize}
\item Respecter les spécif de base
\item Syst secure (le prouver)
\item Choisir une implémentation adaptée en fonction des besoin et de l'état de l'art établie
\end{itemize}
Résultats attendus
\begin{itemize}
\item Etat de l'art le + exhaustif possible
\item Impl. sur CàP et/ou Android
\end{itemize}
\end{itemize}
